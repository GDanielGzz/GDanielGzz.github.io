AMP Classifier

Antimicrobial peptide discriminator on cluster-aware splits, plus a multi-label activity head and a discriminator-in-the-loop mutant designer.

ESM2-650M frozen + MLP head: AUC 0.919 [0.906–0.931], MCC 0.697 [0.667–0.730] — beats XGBoost over 428 engineered features by +5.5 AUC with non-overlapping 95% bootstrap CIs. mmseqs2 clustering at 40% identity for the splits; no test-set iteration.

CRISPR Designer

Trained CRISPR-Cas9 efficiency models head-to-head against the published Rule Set 2 baseline on mammalian and bacterial corpora.

Transformer ties xgboost on Doench 2016 mammalian (ρ 0.420 vs 0.459, overlapping CIs); xgboost wins cleanly on crisprHAL E. coli (ρ 0.626 vs 0.544, non-overlapping CIs, 33,743 sgRNAs). Cross-kingdom transfer learning didn't transfer. Negative results reported honestly.

Valuador MTY

Production Windows desktop application currently used by practicing real-estate appraisers in Monterrey. Three valuation methodologies validated against SHF regulatory standards.

Market homologation, replacement cost, and income capitalization approaches reconciled into a single appraisal report with statistical validation (coefficient-of-variation traffic light, outlier flagging, adjustment caps). ~7,400 LOC Python with ~1,870 LOC of tests including security coverage. PyInstaller + InnoSetup packaged installer; 30-second autosave with 14-day rolling backups; fully offline-first.

Foldit Lite

Browser puzzle game where mutating residues triggers a live ESMFold structure re-prediction, scored as Cα-RMSD to a target backbone.

Background-warmed puzzle queue with asyncio-locked refills so player clicks never wait on 3–15 s ESMFold latency. Three-check validator gates every generated puzzle: endpoint reachable, RMSD gradient usable, solvable within budget.

Python FastAPI React TypeScript 3Dmol.js ESMFold

Nejayote

Browser-based bioprocess simulator for alkaline-ALE on corn-nixtamalization wastewater, every parameter cited from the literature.

Multi-substrate Monod with Cardinal pH/T inhibition (Rosso 1995), Liebig-minimum limitation, DO quasi-steady-state, NH₄⁺ speciation. ALE module with both deterministic phenotypic-margin and stochastic Wright-Fisher / Luria-Delbrück Monte Carlo engines. Custom bounded Nelder-Mead for parameter fitting.

TypeScript React 19 Vite Tailwind

OrganAIzer

Depth-camera box dimensioning and 3D container-loading optimization for moving and transport companies. M.Sc. AI capstone, team of four.

YOLOv8 object detection localizes cardboard boxes in Intel RealSense depth frames; detections fused with depth data give accurate per-box 3D dimensions in the field. A Container Loading Problem solver produces a stepwise physical loading sequence, delivered to operators through a KivyMD cross-platform app with a PostgreSQL backend.
